一、字符集和编码
   在计算机中所有的信息都是以二级制形式存储的，我们在屏幕上看到的英文，字符和文字都是二进制转化之后的结果。通俗来讲按照某种规则将字符存储在计算
机中称之为编码，而将计算机中存储的二进制数字解析出来的过程称之为解码。

   字符集：所有文字，和符号的总称

   编码：就是将字符集转化为计算机可以接受的数字
