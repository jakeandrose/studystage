一、几个简单的常见算法

    1、冒泡排序法（按照从小到大为例）
          冒泡排序法的核心思想：两两之间相互比较，将其中较为小的放在前面，每次比较完后能保证两两之间小的在前大的在后，但是不能保证在最前面的
       是最小的，所以每次都要从头开始。两层循环，第一层循环控制循环次数，循环次数为数组长度减1，内层循环控制两两之间相互比较，每次都从头开始

    2、选择排序法(按照从小到大为例)
          选择排序法的核心思想：从第一个元素开始，所有的元素进行比较，将小的元素放到前面，这样第一次循环后，第一个元素就为最小元素。以此类推。
       两层循环，第一层循环控制循环次数,循环次数为数组长度减1，内层控制元素逐个比较。
